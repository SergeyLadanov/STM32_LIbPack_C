# UTF8_To_CP1251
Библиотека для конвертирования текста кодировки UT8 в CP1251.

## Тестирование работы

Компиляция проекта:
```
make
```
В папке Bin будет расположен исполняемый файл converter.

Перекодирование:
```
./converter [Имя входного файла]
```
Пример:

```
./converter test.txt
```
В каталоге исполняемого файла будет расположен выходной файл out.txt.
